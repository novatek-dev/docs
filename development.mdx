<Info>
  **Prerequisites**:
  - Node.js 18+
  - Docker & Docker Compose
  - 16GB+ RAM
  - Git
</Info>

## Local Setup

### 1. Clone & Install CLI

```bash
# Clone documentation
git clone https://github.com/novatek-dev/docs.git
cd docs

# Install Mintlify CLI globally
npm i -g mint
```

### 2. Run Preview

Navigate to docs directory and start the preview:

```bash
mint dev
```

View documentation at `http://localhost:3000`.

## Docker Development Setup

For full NovaCloud EIP development with containers:

```bash
# Clone main repository
git clone https://github.com/novatek-dev/novacloud-eip.git
cd novacloud-eip

# Copy environment
cp .env.example .env

# Start all containers
docker-compose up -d

# View logs
docker-compose logs -f
```

### Container Ports

| Container | Port | Service |
|-----------|------|---------|
| Next.js | 3000 | Web App |
| Supabase/Postgres | 5432 | Database |
| Ollama | 11434 | LLM API |
| Qdrant | 6333 | Vector DB |
| Neo4j | 7687 | Graph DB |
| Neo4j Browser | 7474 | Graph UI |

## Development Workflow

### Editing Documentation

1. Make changes to `.mdx` files in the docs repo
2. Changes auto-reload at `http://localhost:3000`
3. Commit and push to trigger automatic deployment

### Custom Ports

Run Mintlify on a custom port:

```bash
mint dev --port 3333
```

### Code Quality

Format and validate your documentation:

```bash
# Check for broken links
mint broken-links

# Update CLI
npm update -g mint
```

## Troubleshooting

<AccordionGroup>
  <Accordion title="Port already in use">
    Mintlify will automatically try the next available port. Check which process is using the port:
    ```bash
    lsof -i :3000
    ```
  </Accordion>

  <Accordion title="Docker containers won't start">
    Check system resources and logs:
    ```bash
    docker-compose logs [container-name]
    docker system df
    ```
  </Accordion>

  <Accordion title="Cannot find docs.json">
    Ensure you're in the root directory of the docs repository where `docs.json` is located.
  </Accordion>

  <Accordion title="Node version error">
    Update Node.js to version 18+:
    ```bash
    node --version
    nvm install 18
    nvm use 18
    ```
  </Accordion>
</AccordionGroup>

## Building for Production

```bash
# Generate static build
mint build

# Test production build locally
npm i -g serve
serve -s dist
```
