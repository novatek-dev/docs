# API Anahtarları

API'ye erişim için anahtarlar oluşturun.

## Anahtar Oluştur

1. Ayarlar > API
2. "Yeni Anahtar" tıklayın
3. Adını girin
4. İzinleri seçin
5. Kaydedin

## Anahtar Türleri

- **Okuma Anahtarı**: Sadece veri okuma
- **Yazma Anahtarı**: Veri oluşturma ve güncelleme

## Güvenlik

Anahtarları güvenle saklayın. Herkese göstermeyin.
