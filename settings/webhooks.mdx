# Webhooks

Sistemi dış uygulamalarla integre edin.

## Webhook Oluştur

1. Ayarlar > Webhooks
2. "Yeni Webhook" tıklayın
3. URL'i girin
4. Event'leri seçin
5. Kaydedin

## Desteklenen Events

- İçerik yayınlandı
- İçerik silindi
- Yorum yapıldı
- Kullanıcı eklendi

## Test

Webhook'u test etmek için test event'i gönderin.
